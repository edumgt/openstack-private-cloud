# Python + 금융공학 RAG Lab 커리큘럼

## 1. Lab 개요
- 대상: Python 기초가 있고 금융 도메인 문서를 다루는 엔지니어/연구자
- 형태: 실습 중심 모듈형 커리큘럼
- 목표: 금융 문서 기반 RAG 프로토타입을 구현하고 운영 관점까지 확장

## 2. 모듈 로드맵
| 모듈 | 주제 | 핵심 산출물 |
|---|---|---|
| Module 1 | RAG 기본 구조와 금융 도메인 정의 | 유스케이스 캔버스 |
| Module 2 | 금융 문서 수집/정규화/스키마 설계 | JSONL 데이터셋 |
| Module 3 | 검색기 구현(TF-IDF, 유사도) | Python Retriever |
| Module 4 | 답변 생성 파이프라인 구성 | RAG Pipeline |
| Module 5 | 성능 측정(정확도/근거성/지연시간) | 평가 리포트 |
| Module 6 | 리스크 시나리오(금리/환율/변동성) | 시나리오 질의 세트 |
| Module 7 | 운영 설계(보안/감사/모니터링) | 운영 체크리스트 |
| Module 8 | 캡스톤(사내 데이터 적용) | 데모 및 결과 보고서 |

## 3. 모듈별 상세 커리큘럼

### Module 1. RAG 기본 구조와 금융 유스케이스
- 학습 내용
  - 임베딩/검색/생성 3계층 구조 이해
  - 금융공학 질문 유형 정의(리스크 설명, 상품 비교, 규정 요약)
- 실습
  - `질문 유형 10개`와 `필요 근거 문서` 매핑
- 평가
  - 질문-근거 연결 정확성(질문당 최소 2개 근거 문서 식별)

### Module 2. 데이터 엔지니어링(문서 정규화)
- 학습 내용
  - 시장 리포트, 상품 설명서, 리스크 정책 문서의 표준 스키마 설계
  - 문서 메타데이터(출처, 날짜, 태그) 관리
- 실습
  - JSONL 포맷 금융 문서 30건 이상 생성
- 평가
  - 필수 필드(title, category, content, tags) 누락률 0%

### Module 3. 검색기 구현
- 학습 내용
  - 토큰화, TF-IDF, 코사인 유사도 기초
  - 검색 랭킹 품질 개선(불용어, 태그 가중치)
- 실습
  - `lab/rag_finance/retriever.py` 기반 top-k 검색 구현/비교
- 평가
  - 기준 질의셋에서 top-3 관련 문서 포함률 측정

### Module 4. 답변 생성 파이프라인
- 학습 내용
  - 검색 결과를 근거로 답변 초안 생성
  - 환각 최소화를 위한 근거 우선 템플릿 설계
- 실습
  - `lab/rag_finance/pipeline.py`로 질의-응답 CLI 실행
- 평가
  - 답변 내 근거 문서 ID/제목 명시율 100%

### Module 5. 평가 지표 설계
- 학습 내용
  - Retrieval Precision, Evidence Coverage, Latency 정의
  - 품질 회귀 방지 기준선 수립
- 실습
  - 대표 질의 20개로 평가표 작성
- 평가
  - 지표 결과와 개선 우선순위 제안서 제출

### Module 6. 금융 리스크 시나리오 실습
- 학습 내용
  - 금리 상승/하락, 변동성 급등, 유동성 축소 시나리오
  - 포트폴리오/파생상품 관점의 질의 구성
- 실습
  - 시나리오별 질의 템플릿 5개씩 작성
- 평가
  - 시나리오-답변 일관성 및 근거 적합성 리뷰

### Module 7. 운영 및 거버넌스
- 학습 내용
  - 데이터 접근 제어, 질의 로그, 감사 추적성
  - Private Cloud 환경 배포 고려사항
- 실습
  - 운영 체크리스트 작성(보안/권한/모니터링)
- 평가
  - 운영 리스크 항목 누락 없이 체크리스트 완성

### Module 8. 캡스톤
- 학습 내용
  - 사내 데이터 반영 및 데모 시나리오 구성
  - 이해관계자 대상 결과 설명
- 실습
  - "질문 → 검색 근거 → 답변 초안" 데모 발표
- 평가
  - 정확도/재현성/운영성 관점 종합 점검

## 4. 권장 운영 규칙
- 모든 답변은 최소 2개 이상 근거 문서와 함께 제시
- 데이터셋 변경 시 기준 질의셋으로 회귀 평가 수행
- 금융 리스크 관련 답변은 최종 의사결정 자료가 아님을 명시
