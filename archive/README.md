# Archive

이 폴더는 현재 20강 학습 흐름에 직접 사용되지 않는 레거시 산출물을 보관합니다.

- `legacy/results/`: 과거 45강 검증 로그/캡처 및 보안 실험 결과
- `legacy/curriculum/`: 과거 심화 트랙 메타데이터
- `legacy/scripts/`: 레거시 결과 캡처 생성 스크립트

현재 학습에 필요한 실행 경로는 루트의 `lecture01~lecture20`, `ansible/`, `docs/`, `scripts/`입니다.
